Software & Tools Used

For this project, we used a combination of tools and technologies to design, build, and test our website.

Frontend Development

We used HTML, CSS, and a bit of JavaScript to build the structure, design, and interactive features of our website and python.

  • HTML was used to create the layout and content of each page.
  • CSS was used to style the website, including colours, layout, and effects such as glowing inputs.
  • JavaScript was used to add movement and animation, including the walking character on the screen.
  • Linux was used for python and Windows was used for the rest of the project

Development Tools

  • Visual Studio Code: Our main coding environment which allowed us to write, edit, and organise our code efficiently.
  • Web Browsers: Tools like Google Chrome to test and view how our website looks and behaves in real time.

Design & Assets

For visuals and design elements, we created and edited images for our animated character. These images were used to create a walking animation by switching between frames.

We also focused on choosing colours, layouts, and simple design elements to make the website look clean and user-friendly.

Learning Resources

We used different online resources to help us with this project:

  • YouTube & Google: For step-by-step tutorials and finding answers when we got stuck.
  • AI Tools: To help explain code and generate ideas. While we don’t understand everything completely yet, these tools helped us learn and make progress.
  • Reddit: To see how other people solved similar problems and to learn from their experiences.
